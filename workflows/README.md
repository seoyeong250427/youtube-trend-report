# Workflows

여기는 "업무 매뉴얼"이 저장되는 곳입니다. 반복되는 업무가 생기면, 그 업무를 처리하는 단계별 지시서를 이 폴더에 파일로 만듭니다.

각 Workflow 파일은 다음을 포함해야 합니다:

- **목표** — 이 업무가 뭘 달성해야 하는지
- **입력값** — 시작하기 전에 필요한 정보
- **사용할 Tool** — `tools/` 폴더의 어떤 파일을 쓰는지
- **결과물** — 끝났을 때 어떤 모습이어야 하는지
- **문제 발생 시** — 흔한 에러와 대처법

아직 Workflow가 없습니다. 반복되는 업무가 처음 생기면 여기에 추가됩니다.
