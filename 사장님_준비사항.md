# 사장님이 하실 일 (딱 이것만 하면 끝)

이 자동화가 매주 일요일 저녁 8시에 돌아가려면, 열쇠 4개를 발급받아서 claude.ai에 등록하는 것만 하시면 됩니다. 하나씩 순서대로 하시면 됩니다. 각 단계는 5분 안팎입니다.

체크박스에 표시하면서 진행하세요.

---

## 1. YouTube API 키 발급

- [ ] [console.cloud.google.com](https://console.cloud.google.com) 접속 → 로그인
- [ ] 상단의 프로젝트 선택 → "새 프로젝트" → 이름 아무거나 입력 → 만들기
- [ ] 왼쪽 메뉴 "API 및 서비스" → "라이브러리" 클릭
- [ ] 검색창에 "YouTube Data API v3" 입력 → 클릭 → **활성화** 버튼
- [ ] 왼쪽 메뉴 "API 및 서비스" → "사용자 인증 정보" 클릭
- [ ] "+ 사용자 인증 정보 만들기" → "API 키" 선택
- [ ] 생성된 키(문자+숫자 조합)를 메모장에 복사해두기 → 이게 **YOUTUBE_API_KEY**

## 2. Gmail 앱 비밀번호 발급

- [ ] [myaccount.google.com/security](https://myaccount.google.com/security) 접속
- [ ] "2단계 인증"이 꺼져 있으면 먼저 켜기 (이미 켜져 있으면 통과)
- [ ] 페이지 상단 검색창에 "앱 비밀번호" 입력 → 클릭
- [ ] 앱 이름에 아무거나 입력(예: "리포트봇") → "만들기"
- [ ] 화면에 나온 16자리 코드를 복사 (공백 있어도 상관없음) → 이게 **EMAIL_APP_PASSWORD**
- [ ] 로그인한 Gmail 주소 자체를 메모 → 이게 **EMAIL_ADDRESS** (그리고 리포트도 이 주소로 받게 하려면 **REPORT_RECIPIENT_EMAIL**도 같은 주소)

## 3. Notion 토큰 + 데이터베이스 준비

- [ ] [notion.so/my-integrations](https://notion.so/my-integrations) 접속 → 로그인
- [ ] "New integration" 클릭 → 이름 입력(예: "트렌드리포트") → 제출
- [ ] 생성된 화면에서 토큰(`secret_`으로 시작하는 문자열) 복사 → 이게 **NOTION_TOKEN**
- [ ] 노션 앱으로 가서, 리포트 데이터를 쌓을 새 페이지(데이터베이스/표) 하나 만들기
- [ ] 그 페이지 우측 상단 "···" 메뉴 → "Connections" → 방금 만든 integration 검색해서 연결
- [ ] 그 페이지의 주소창 URL을 보면 `...notion.so/제목-32자리문자열?v=...` 형태입니다. `-` 뒤, `?` 앞의 32자리 문자열을 복사 → 이게 **NOTION_DATABASE_ID**

## 4. claude.ai에 등록

- [ ] claude.ai 접속 → 우측 상단 프로필 아이콘 → **Settings**
- [ ] **Environments** 메뉴 클릭
- [ ] 기존에 Routine 만들 때 생긴 Environment가 있으면 그걸 편집, 없으면 "New environment"
- [ ] **Environment variables**에 아래 6개를 하나씩 이름=값으로 추가:

| 이름 | 값 |
|---|---|
| `YOUTUBE_API_KEY` | 1번에서 복사한 값 |
| `EMAIL_ADDRESS` | 2번에서 메모한 Gmail 주소 |
| `EMAIL_APP_PASSWORD` | 2번에서 복사한 16자리 코드 |
| `REPORT_RECIPIENT_EMAIL` | 리포트 받을 이메일 (본인 주소면 됨) |
| `NOTION_TOKEN` | 3번에서 복사한 토큰 |
| `NOTION_DATABASE_ID` | 3번에서 복사한 32자리 ID |

- [ ] 저장

## 5. Routine을 이 저장소에 연결

- [ ] Routine 만드는 화면(또는 기존 Routine 편집)에서, 저장소를 **`seoyeong250427/youtube-trend-report`**로 지정
- [ ] 방금 만든 Environment를 선택
- [ ] 실행 프롬프트에 "이 저장소의 `workflows/weekly_youtube_trend_report.md`를 따라서 실행해줘" 정도로 입력
- [ ] 일정은 매주 일요일 20:00으로 설정
- [ ] "지금 실행" 버튼으로 한 번 테스트 실행 → 이메일이 실제로 오는지, 노션에 새 줄이 생겼는지 확인
